About Project

Beauty Bomber is an E-commerce website based in india and first unicorn statup by an indian women(cloned of nykaa)

Collaborative Project made in 5 days with a team of 4 full stack developer

    Tech Stack used is :
  • HTML,CSS and JavaScript for frontend
  • Node and Express for building server
  • Mongo DB and Mongoose
  • Libraries like splide js, JWT,Google Oath 2.0 , bcrypt for hashing and someother libraries.
    Key Features:
  • Login,Signup ,forget password and Authenttication with google OATH 2.0, brcypt and JWT
  • Add To Cart functionality and remove from cart With api end points created with Node,express and mongo
  • Fully Reusable Product page which you can see on bottom side (for evey product it is working great)
  • Stripe Payment Gateway Integration with node,express and mongodb
  • Seller Page from where you can add product to the website and can easily sell
    Areas Of responsibility
  • Authenttication with bcrypt ,Google OATH and OTP hashing .
  • Fully reusable product page plus integration with payment Gateway
  • Stripe Payment Gateway build with stripe library and Node express
